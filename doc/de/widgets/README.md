# Widget-Katalog

[Anwenderhandbuch](../README.md) · [English](../../en/widgets/README.md)

Widget nach Aufgabe wählen. Mehrere editor-sichtbare Varianten bleiben bewusst
getrennt, damit jedes Widget eindeutig auffindbar und konfigurierbar ist.

Historische Template-IDs bleiben für stabile Projekte erhalten. Schreibweisen
wie `Adition`, `Vuetify` oder `view-in-widget8` bezeichnen keine aktuelle
Abhängigkeit und sollten nicht geändert werden.

## Schalten und navigieren

- [Buttons](buttons.md) – View öffnen, Link aufrufen oder State schreiben
- [Buttons vertikal](buttons-vertical.md) – gleiche Aktionen, vertikales Layout
- [Icon-Buttons](icon-buttons.md) – kompakte Aktionen nur mit Icon
- [Checkbox](checkbox.md) – booleschen oder eigenen Aus-/Ein-Wert schalten
- [Switch](switch.md) – Zustand als Schalter bedienen

## Werte eingeben

- [Eingabe](input.md) – Text oder Zahl eingeben
- [Select und Autocomplete](select.md) – Wert aus einer Liste wählen
- [Slider](slider.md) – numerischen Wert horizontal oder vertikal ändern
- [Runder Slider](slider-round.md) – numerischen Wert kreisförmig ändern

## Werte und Inhalte anzeigen

- [Wertanzeige](value.md) – einzelnen State formatiert anzeigen
- [HTML Card](html-card.md) – Titel, Text, Bild und Aktion kombinieren
- [Fortschritt](progress.md) – Wert als linearen Fortschritt anzeigen
- [Fortschritt kreisförmig](progress-circular.md) – Wert kreisförmig anzeigen
- [Material-Design-Icons und Hilfswidgets](material-design-icons-and-images.md)

## Daten darstellen

- [Liste](list.md) – Zeilen mit Text, Buttons, Switches oder Checkboxen
- [IconList](iconlist.md) – zustandsabhängige Icons als Raster
- [Tabelle](table.md) – JSON-Zeilen in Spalten
- [Diagramme](charts.md) – Balken, Kreis, JSON oder Linienverlauf
- [Kalender](calendar.md) – JSON-Termine in Monat, Woche oder Tag
- [Warnmeldungen](alerts.md) – Meldungswarteschlange anzeigen und quittieren
