# Input, Select and Autocomplete

[Back to README](../../../README.md#widget-documentation)

Three native VIS 2 controls for entering text/numbers or choosing values.

Template ids: `tplVis2-materialdesign-Input`,
`tplVis2-materialdesign-Select` and `tplVis2-materialdesign-Autocomplete`.

<img src="../../media/vis2_input_runtime.png" alt="Material Design input controls in VIS 2">

## Editor settings

<table>
<tr><td><img src="../../media/vis2_input_editor_overview.png" width="300"></td>
<td><ul><li><b>Input:</b> text/number type, mask, validation, counter and clear icon.</li><li><b>Select:</b> boolean, value list, JSON string or JSON object data.</li><li><b>Autocomplete:</b> uses the Select data methods and filters entries while typing.</li><li>Set labels, hints, icons and colors in their optional groups.</li></ul></td></tr>
</table>

Value-list example: values `0;1;2`, labels `Off;Auto;On`. JSON entries can use
`value`, `label`, `subLabel` and `icon`.
