# Widget catalog

[User guide](../README.md) · [Deutsch](../../de/widgets/README.md)

Choose a widget by task. Editor-visible variants deliberately remain separate so
that every widget is easy to find and configure.

Historical template IDs remain stable for existing projects. Names such as
`Adition`, `Vuetify` or `view-in-widget8` do not describe current dependencies
and must not be changed.

## Switch and navigate

- [Buttons](buttons.md) – open a view or link, or write a state
- [Buttons Vertical](buttons-vertical.md) – same actions with vertical layout
- [Icon Buttons](icon-buttons.md) – compact icon-only actions
- [Checkbox](checkbox.md) – switch boolean or custom off/on values
- [Switch](switch.md) – operate a state as a switch

## Enter values

- [Input](input.md) – enter text or numbers
- [Select and Autocomplete](select.md) – choose a value from a list
- [Slider](slider.md) – change a numeric value horizontally or vertically
- [Slider Round](slider-round.md) – change a numeric value on a circular control

## Display values and content

- [Value](value.md) – display one formatted state
- [HTML Card](html-card.md) – combine title, text, image and action
- [Progress](progress.md) – display a value as linear progress
- [Progress Circular](progress-circular.md) – display a value as circular progress
- [Material Design Icons and Utilities](material-design-icons-and-images.md)

## Display data

- [List](list.md) – rows with text, buttons, switches or checkboxes
- [IconList](iconlist.md) – state-dependent icons in a grid
- [Table](table.md) – JSON rows in columns
- [Charts](charts.md) – bar, pie, JSON or line history
- [Calendar](calendar.md) – JSON events in month, week or day views
- [Alerts](alerts.md) – display and acknowledge a message queue

## Views and layout

- [Top App Bar](top-app-bar.md) – navigation with app bar and drawer
- [Responsive Layout](responsive-layout.md) – multiple views in masonry or grid
- [Advanced View in Widget](html-widgets.md) – select one child view by state
- [Dialog](dialog.md) – open a view or iFrame in a dialog
